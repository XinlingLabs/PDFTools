# PDF Tools
使用说明：
1. 选择文件<br>
   选择或拖入 PDF 文件，单文件用于拆分。
   一次选择多个文件时，用于批量重命名。

2. 规则拆分<br>
   只输入一个数字，例如 3。
   程序会按每 3 页一直拆到 PDF 结束。
   每段期望页数都按手动输入的 3 计算。

3. 不规则拆分<br>
   输入多个数字，例如 2,3,5。
   也可以逐行输入 2、3、5。
   期望页数只取手动输入的每个数字。

4. 页数校验<br>
   实际页数与期望页数不一致时会提示。
   日志标红、进度条变红，并弹窗提醒。

5. 文件命名<br>
   新文件名可选，支持逐行或逗号分隔。
   名称数量必须与生成文件数量一致。
   文件名不能重复，已存在文件不会覆盖。

6. 输出设置<br>
   输出路径为空时，默认使用当前目录。
   路径不存在时，程序会自动创建。
   勾选输出 CSV 后，才生成拆分结果.csv。
